# 🌿 Real HD Decorative Base Game

**Real HD Decorative Base Game** is a high-definition visual replacement pack for Factorio’s vanilla decorative world details.

It upgrades the base-game rocks, mud decals, grass clumps, bushes, stumps, dead plants, enemy decals, ground clutter, and natural decorative sprites with cleaner HD textures while keeping the original Factorio style.

The goal is simple:

**Sharper world detail. Cleaner natural textures. Same vanilla gameplay.**

---

## 🖼️ What It Does

Real HD Decorative Base Game replaces many of Factorio’s base-game decorative sprites with cleaner, sharper HD versions.

This includes natural map detail such as rocks, cracked mud, dry grass, red desert plants, stumps, enemy decals, and small ground details that make Nauvis feel more alive.

The mod does not add new entities, recipes, technologies, items, enemies, or mechanics. It is a visual texture replacement only.

Everything still behaves exactly like vanilla Factorio.

## 🎯 Design Goal

Real HD Decorative Base Game is made for players who want Factorio to look cleaner and sharper without changing how the game plays.

The goal is **not** to make Factorio look like a different game.

The goal is to keep the original vanilla look, but improve the texture quality so the world feels more detailed at higher resolutions.

Real HD Decorative Base Game is built around:

* vanilla-style colour and shape
* cleaner 2x texture detail
* sharper world decoration
* improved natural terrain clutter
* better-looking rocks and decals
* safe visual replacement only

## ⚙️ Performance Notes

This is a texture replacement mod.

It does not run scripts during gameplay and does not add UPS-heavy logic.

Performance impact depends mostly on your GPU, VRAM, texture quality settings, and how many HD texture packs you use at the same time.

For best results:

* use high texture quality in Factorio settings
* keep texture compression enabled if you need lower VRAM use
* avoid stacking too many large HD packs on weaker systems
* restart Factorio after changing texture packs

## 🔗 Compatibility

Real HD Decorative Base Game should work with most mods because it is visual-only.

It is designed to be safe with normal gameplay mods, overhaul mods, and Space Age.

### Works well with:

* vanilla Factorio
* Space Age
* visual mods
* terrain mods
* quality-of-life mods
* most overhaul mods

### Possible conflicts:

Other mods that replace the exact same decorative textures may override or be overridden depending on load order.

If another texture pack changes the same decorative sprites, whichever mod loads last may control the final look.

## ❓ FAQ

### Does this change gameplay?

No.

This is visual-only.

### Does this affect UPS?

No gameplay script runs during normal play.

### Can I add it to an existing save?

Yes.

### Can I remove it later?

Yes. Saves should continue loading without it.

### Is this AI upscaled?

The textures are HD replacement versions made to stay close to the original Factorio style. The goal is cleaner vanilla-style detail, not a full art-style replacement.

### Why is this separate from other Real HD packs?

To keep file sizes manageable and let players choose which parts of Factorio they want upgraded.
